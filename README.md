## 🚀 部署说明

本网站使用 GitHub Pages 自动部署，当代码推送到 `main` 分支后，GitHub 会自动构建和部署网站。

### 本地开发

如果需要本地预览：

1. 克隆仓库
   ```bash
   git clone https://github.com/de-bao/de-bao.github.io.git
   cd de-bao.github.io
   ```

2. 直接在浏览器中打开 `index.html` 文件即可预览

### 更新简历

1. 修改 `index.html` 文件中的内容
2. 提交更改
   ```bash
   git add index.html
   git commit -m "更新简历信息"
   git push origin main
   ```

3. 等待几分钟，GitHub Pages 会自动更新网站

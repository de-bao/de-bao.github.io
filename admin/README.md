# 后台管理页面

## 📍 访问地址

部署后可通过以下地址访问：
- `https://debao.qzz.io/admin/`
- `https://de-bao.github.io/admin/`

## 🔧 配置说明

- **API 地址**: `https://api.debao.qzz.io/api`
- **登录用户名**: `admin`
- **登录密码**: 任意（当前为简化版认证）

## ✨ 功能

- 文章 CRUD 操作
- 文章发布（生成 Jekyll Markdown）
- 深色主题界面
- 响应式设计

## 📝 使用说明

1. 访问 `/admin/` 页面
2. 使用 `admin` 用户名登录
3. 开始管理你的博客文章

详细使用说明请参考项目根目录的 `API_TEST_RESULTS.md`
